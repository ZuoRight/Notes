# 引言

根据图灵机和冯诺伊曼结构，计算机必须具备五大基本组成部件

- 输入设备：装载数据和程序
- 存储器：记住程序和数据
- 运算器：完成数据加工处理
- 控制器：控制程序执行
- 输出设备：显示处理结果

## 主板

早期主板有两块比较重要的芯片：北桥和南桥。

北桥靠近CPU，负责与CPU、内存、显卡等高速设备通信，并与南桥连接

南桥负责与I/O（SATA、USB、LAN）、硬盘等低速设备通信

> SATA 一般指串行总线，负责主板和大容量存储装置（硬盘、光驱）之间传输数据
