# 性能指标

## 带宽 bandwidth

指单位时间内信道能传输的最大数据量

- 电气领域：指的是模拟带宽，单位：Hz(赫兹)
- 计算机领域：指的是数字带宽，单位：bps(比特率) 或 Bps(字节率)
  > 十进制：用K、M、G等表示，1K=1000，比如1Kbps，表示1000比特每秒  
  > 二进制：用Ki、Mi、Gi等表示，1Ki=1024，比如1KiBps，表示1024字节每秒

带宽的限制：信噪比，信道传输的有用信息与无用的干扰噪声的比值，单位为分贝(dB)

## 吞吐量 throughput

指单位时间内信道的通过的实际数据量
