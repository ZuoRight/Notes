# 类文件读写

## StringIO：操作缓存中的str数据

```python
from io import StringIO

output_buffer = StringIO()
# 还可以初始化
# output_buffer = StringIO('Hello!\nHi!\nGoodbye!')
```

## BytesIO：操作缓存中的二进制数据

```python
from io import BytesIO

output_buffer = BytesIO()
# 还可以初始化
# output_buffer = BytesIO(b'\xe4\xb8\xad\xe6\x96\x87')
```

获取缓存中的数据

```python
print(f.getvalue())
```
