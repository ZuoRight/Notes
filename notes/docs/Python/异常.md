# 异常

```python
try:
    pass  # 可能会出错的地方
except Xxx1Error as e1:
    print(e1)
except Xxx2Error as e2:
    print(e2)
except (Xxx3Error,Xxx4Error) as es:
    print("可以同时捕获多个异常")
except Exception as e:
    print('通用异常，可匹配任意异常')
except:
    print("如果不写异常名称，将匹配所有异常")
else:
    print("如果没有异常则执行")

finally:
    print("不管有没有异常最后都会执行")
```
