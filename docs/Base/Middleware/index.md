# 引言

中间件是位于操作系统和应用程序之间的软件层，用于提供复杂的功能服务，使得开发人员可以更容易地实现和集成不同的技术组件。

比如

- Nginx 作为反向代理和负载均衡器
- Redis 用于缓存和消息队列
- RabbitMQ、Kafka 消息队列，用于消息传递和异步通信
- Elasticsearch 用于日志管理和分析
- Keycloak 管理身份验证和授权
- Kong 管理和保护API
- OAuth 2.0 身份验证
- Tomcat 应用服务器，用于运行和管理 Web 应用和企业级应用
