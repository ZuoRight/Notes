# [ZuoRight](http://zuoright.com)

## VS Code

### Settings Sync

- 上传：`Shift + Alt/Opt + U`
- 下载：`Shift + Alt/Opt + D`

### PicGo

- 从剪切板上传：`Ctrl/Cmd + Alt/Opt + U`
- 从文件夹上传：`Ctrl/Cmd + Alt/Opt + E`
- 输入文件路径上传：`Ctrl/Cmd + Alt/Opt + O`
